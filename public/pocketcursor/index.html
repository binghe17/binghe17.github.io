🌳 PocketCursor란?

PocketCursor는 macOS Cursor IDE를 텔레그램에서 완전히 제어할 수 있는 개인 AI 에이전트입니다.

Cursor 작업 내용이 실시간으로 텔레그램에 전달되며, 어디서든 명령을 내릴 수 있습니다.

⚡ 주요 기능
🤖 로컬 LLM (Gemma 4 등) 연동
💬 Cursor 채팅 실시간 포워딩
🦾 macOS UI 자동화 (Playwright)
🖼 AI 이미지 생성 (ComfyUI/DrawThings)
🔊 TTS/STT 음성 지원 (한국어)
📊 시스템 모니터링 (CPU/RAM/배터리)
🔒 다중 기기 보안 인증
🧬 자기개선 메타인지 시스템
